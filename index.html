Which file should you upload? Only the AIS JSON or TIS JSON — not the PDF, not the CSV, and not the ACF (feedback) file.
  1. Log in to incometax.gov.in
  2. Click Services → Annual Information Statement (AIS)
  3. Click Proceed → you'll be redirected to the AIS portal
  4. Select the financial year, then click the AIS tile
  5. Click the download icon (↓) and choose AIS – JSON (not PDF, not ACF)
  6. The downloaded file will be a .json file — upload that here

Common mistake: The ACF (feedback) file and the AIS PDF are not JSON files this viewer can read. Make sure you select JSON format in step 5.
Your data stays on your device. All parsing happens in your browser using local JavaScript. No file content is sent to any server or third party at any point.
